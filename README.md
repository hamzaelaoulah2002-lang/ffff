# Taquin Solver in Python

## Description

Ce projet implémente le jeu du Taquin (8-puzzle) en Python en utilisant la bibliothèque NumPy.

Le programme permet de :

- Générer automatiquement une grille de Taquin aléatoire.
- Représenter l'état du plateau sous forme de matrice.
- Déterminer les mouvements possibles de la case vide.
- Mélanger le plateau tout en garantissant une configuration valide.
- Vérifier si la configuration finale (objectif) est atteinte.

## Technologies utilisées

- Python
- NumPy
- Google Colab / Jupyter Notebook

## Fonctionnalités

- Gestion dynamique de la taille du plateau.
- Génération d'états aléatoires.
- Affichage formaté du puzzle.
- Recherche des déplacements possibles.
- Validation de l'état final.
